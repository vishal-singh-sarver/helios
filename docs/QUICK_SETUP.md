# HeliosGUI - Complete Setup & Deployment Guide

**Everything you need to set up and run HeliosGUI from GitHub in one page.**

---

## 🚀 For New Users - Quick Start (15 minutes)

### 1️⃣ Prerequisites

Verify you have installed:
```bash
node --version        # Must be v22.0.0 or higher
npm --version         # Must be v10.0.0 or higher
python3 --version     # Must be v3.10 or higher
git --version         # Any version
```

**macOS Only:**
```bash
xcode-select --install  # Install Xcode Command Line Tools
```

**Linux Only:**
```bash
sudo apt-get install build-essential python3-dev
```

### 2️⃣ Clone & Setup

```bash
# Clone repository
git clone https://github.com/YOUR_USERNAME/helios-gui.git
cd helios-gui

# Install dependencies
npm install

# Build frontend + backend
npm run build

# Run development server
npm run dev
```

**That's it!** Your app is running:
- 🌐 UI: Open locally in the Electron window
- 🔌 Backend: Running on http://127.0.0.1:8008

## 📖 Common Commands

### Development

| Command | What It Does |
|---------|-------------|
| `npm run dev` | Start dev server (auto hot-reload) |
| `npm run build` | Build frontend + backend |
| `npm test` | Run tests |
| `npm run lint` | Check code quality |
| `npm run format` | Format code |

### Packaging (Create Installers)

| Command | Creates |
|---------|---------|
| `npm run package:mac` | macOS: `.dmg` and `.pkg` files in `dist/` |
| `npm run package:win` | Windows: `.exe` file in `dist/` |
| `npm run package:linux` | Linux: `.AppImage` and `.deb` files in `dist/` |
| `npm run package:all` | All platforms |

```bash
npm run dist:clean       # Remove build artifacts
npm run dist:clean-all   # Remove everything (needs npm install again)
```

```

❌ dist/                      # 200 MB - generated by npm run package

❌ backend-api/dist/          # 200 MB - PyInstaller output
❌ resources/backend/         # 200 MB - Backend binaries
❌ .env                       # Secrets/API keys
❌ *.db, *.log                # User data
❌ .DS_Store, Thumbs.db       # OS files
```

If you accidentally added any of these, remove with:
```bash
git rm --cached <filename> -r
git commit -m "Remove build artifacts"
```

## 🐛 Troubleshooting

### Backend Won't Start

```bash
# Check logs
cat ~/Library/Application\ Support/Helios/logs/backend.log

# Verify Python (must be 3.10+)
python3 --version

# Rebuild backend
npm run sync-backend

# Complete reset
npm run dist:clean-all
npm install
npm run build
npm run dev
```

### Port 8008 Already in Use

```bash
# macOS/Linux: Kill process
lsof -i :8008 | awk 'NR!=1 {print $2}' | xargs kill -9

# Windows: Check and kill
netstat -ano | findstr :8008
taskkill /PID <PID> /F
```

### npm install Fails

```bash
npm cache clean --force
npm install
```
