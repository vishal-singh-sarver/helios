Helios - macOS Installation

System Requirements:
- macOS 10.12 (Sierra) or later
- Intel or Apple Silicon processor
- 200 MB free disk space
- Internet connection for initial setup

Installation Process:
1. Review and accept the license agreement
2. Choose installation options
3. Installation will complete automatically
4. Application will be available in Applications folder

After Installation:
- Helios will be installed in /Applications/
- You can launch it from Spotlight or Applications folder
- The application requires admin permissions for first-time setup
- Backend services will start automatically

For assistance with installation, please contact support.
