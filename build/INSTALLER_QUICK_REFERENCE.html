⚙️ Key Configuration Summary

Setting Value Purpose
oneClick false Enable multi-step UI (not one-click install)
license build/license.txt Show EULA in Step 2 (mandatory acceptance)
allowToChangeInstallationDirectory true Allow user to choose install path
perMachine true Install for all users (not current user only)
allowElevation true Allow UAC elevation (admin prompt on Windows)
runAfterFinish true Auto-launch app after installation completes
createDesktopShortcut true Create shortcut on user's desktop

📋 What You Need to Do Now

CRITICAL: Create build/icon.ico

Why: The installer uses this for branding (window icon, shortcuts, Add/Remove Programs)

How:

  1. Export your app logo as PNG (256×256 pixels)
  2. Convert to ICO format using:
    # Option A: Online (iconconverter.com) # Option B: ImageMagick convert logo.png -define icon:auto-resize=256,128,64,32,16 icon.ico # Option C: Photoshop/Inkscape # Export → ICO format
  3. Save as: /Users/navyug/helios_gui/build/icon.ico

Then: Run the Build

$ npm install # (if needed) $ npm run build # Build app files $ npm run package:win # Create installer

Finally: Test the Installer

$ dist/Helios-1.0.0-setup.exe
